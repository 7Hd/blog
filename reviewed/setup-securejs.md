
Download securejs from [Arxan Developer](https://developer.arxan.com/applications).

# Mac

* Setup Application
    * update `~/.bash_profile`
      ```
      ...
      export PATH= "/PATH/TO/SECUREJS/bin:$PATH"
      ```
      ```bash
      source ~/.bash_profile
      ```
    * set access permissions
      ```sh
      chmod -R 777 /PATH/TO/SECUREJS
      ```
